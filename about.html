About The Web Page

Developers: Adam Schuhknecht and Sam Ryan

Description of the Project

This web page serves as an informtional website about a fictional "dream team" which features many of the best NBA players from the past and present. There are five html pages that can be navigated:

  1. The Home Page (index.html)
  2. The About Page (about.html)
  3. The Upcoming Events Page (catalog.html)
  4. The Roster Page (roster.html)
The website was built using html and css, along with the bootstrap framework. Javascript and JQuery were also used for additional functionality of the website, such as filtering the games on the upcoming games page.
